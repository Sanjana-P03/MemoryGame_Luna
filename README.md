# MemoryGame_Luna
A story-driven browser puzzle game where players help Luna recover her lost memories through mazes, memory challenges, and interactive storytelling.
Play Online :
   https://sanjana-p03.github.io/MemoryGame_Luna/

**Features**
Story-based gameplay
Maze and memory puzzles
Multiple chapters and challenges
Interactive dialogues and animations
Progress saving using LocalStorage
Responsive browser-based design

**Technologies Used**
HTML5
CSS3
JavaScript (ES6)
JSON
LocalStorage API
HTML5 Audio API
